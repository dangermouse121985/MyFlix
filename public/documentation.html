My Flix API Documentation

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
API Functions Description URL Endpoints Descriptions
Return All Movies GET /movies Return a list of ALL movies with their id, title, and URL (URL used to retrieve detailed information about the movie)
Return One Movie by Title GET /movies/{movie-title} Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
Return All Genre GET /genres Return a list of all genre including their description
Return One Genre by Name GET /genres/{genre-title} Return data about a genre (description) by name/title (e.g., “Thriller”)
Return All Directors GET /directors Return a list of all directors including (bio, birth year, death year)
Return One Director by Name GET /directors/{director-name} Return data about a director (bio, birth year, death year) by name
Return All Actors GET /actors Return list of actors including their data (bio, birth year, death year)
Return One Actor by Name GET /actors/{actor-name} Return data about an actor (bio, birth year, death year) by name
Return All Users GET /users Return a list of All Users including their data (username, password, first name, last name, email, birthdate)
Return One User by Username GET /users/{username} Return one user by name including their data (username, password, first name, last name, email, birthdate)
Create a New User POST /users Allow new users to register
Update User Info PUT /users/{username} Allow users to update their user info
Add Movie to User's Favorites PUT /users/{username}/favorites/{movie-id} Allow users to add a movie to their list of favorites (showing only a text that a movie has been added)
Remove a Movie from User's Favorites DELETE /users/{username}/favorites/{movie-id} Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed)
Delete User DELETE /users/{username} Allow existing users to deregister (showing only a text that a user email has been removed)
